# Älyvalohain ohjauskeskus

Ohjauskeskuksen avulla voidaan tarkastella ja ohjata tärkeimpiä toimintoja. Käynnistä `main.py` käyttääksesi ohjelmaa.  
_Esittelytarkoituksessa osa arvoista on satunnaisia._

### 🦈 Infonäyttö

**Perustiedot**:

- Valojen määrä yhteensä
- Päällä olevien valojen määrä
- Palaneiden / rikkinäisten valojen määrä
- Energiankulutus (kW)
- Kunnossa olevat valot

**Huoneiden tila:**

- Varatut huoneet yhteensä (huonekortti paikallaan)
- Pienet huoneet varattuna
- Isot huoneet varattuna

### 🦈 Valikko

1. Näytä tilanne (näyttää järjestelmän tilan)
2. Päivitä järjestelmä (arpoo uudet numerot esittelytarkoituksessa)
3. Aktivoi hätävalaistus (simuloi sähkökatkon, aktivoi käytävän hätävalaistuksen ja rajoittaa kirkkauden 10%:iin)
4. Poistu (sulkee ohjelman)

### 🦈 Valitse toiminto

Käyttäjä voi syöttää numeron (1-4) valitakseen haluamansa toiminnon.

### 🦈 Testaus

Kaikki valikon toiminnot ovat testattu ja todettu toimiviksi. Virheellinen syöte antaa virheilmoituksen.
